# Автотестирование сайта Stellar Burgers

**URL сайта:** https://stellarburgers.education-services.ru/

## Общая информация

Набор автоматизированных тестов для проверки функционала веб‑приложения *Stellar Burgers*. Все тесты расположены в директории `tests/`.

## Структура тестовых сценариев

### 1. `test_registration.py`
Проверяет регистрацию нового пользователя по критериям:
- поле **«Имя»** не должно быть пустым;
- поле **Email** должно содержать валидный адрес в формате `логин@домен` (пример: `123@ya.ru`);
- **пароль** — не менее 6 символов.

**Источник данных:** файл `loghelper.py` (генерация для регистрации).

### 2. `test_login.py`
Тестирует вход в аккаунт через разные интерфейсы:
- кнопка **«Войти в аккаунт»** на главной странице;
- раздел **«Личный кабинет»**;
- форма регистрации (кнопка **«Войти»**);
- форма восстановления пароля (кнопка **«Войти»**).

### 3. `test_incorrect_password.py`
Проверяет отображение сообщения об ошибке при вводе некорректного пароля.

**Используемые файлы:**  
- `data.py`  
- `locators.py`  
- `curl.py`

### 4. `test_personal_account.py`
Тестирует переходы по клику на **«Личный кабинет»** для авторизованных пользователей

**Используемые файлы:**  
- `data.py`  
- `locators.py`  
- `curl.py`

### 5. `test_go_to_constructor.py`
Проверяет переход в конструктор бургеров:
- по кнопке **«Конструктор»** в профиле;
- по клику на логотип сайта.

### 6. `test_log_out.py`
Тестирует выход из аккаунта для авторизованного пользователя.

### 7. `test_constructor_section.py`
Проверяет переключение разделов в конструкторе:
- параметризованный тест для вкладок **«Соусы»** и **«Начинки»**;
- отдельный тест для вкладки **«Булки»** (с предусловием: активна вкладка **«Соусы»**).


## Вспомогательные файлы

### `conftest.py`
Содержит фикстуры:
- настройка браузера;
- авторизация;

**Используемые переменные:**  
- `data.py`  
- `locators.py`  
- `curl.py`

### `curl.py`
Хранит основные URL сайта.

### `data.py`
Содержит тестовые данные авторизации:
- логин;
- имя;
- пароль.

### `loghelper.py`
Скрипт для генерации случайных данных (используется в тестах регистрации).

### `locators.py`
Локаторы элементов страницы для взаимодействия в тестах.

## Примечания
- Все тесты написаны с использованием **Selenium** и **pytest**.
- Для запуска требуется настроенное виртуальное окружение (`venv`) и установленные зависимости.
- Локаторы элементов и тестовые данные вынесены в отдельные файлы для удобства поддержки и апдейта.
